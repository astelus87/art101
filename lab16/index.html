Lab 16 - JSON & APIs

Challenge

For this lab, we were challenged to put in a specific API for the comic XKCD onto our website. The end goal was to display the current comic onto our websites.

Problems

For some reason, there was some kind of error that kept displaying on our console. We found out later on that we needed to get a chrome extension called CORS for the output to display properly on our webpage.

Reflection

My friends and I ended up working on this project together, and after looking back into the lecture videos, we were able to figure it out.
